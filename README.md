# Nil UI

A dark theme for [Zed](https://zed.dev) editor.

## Installation

### From the extension registry

1. Open Zed
2. Open the command palette (`cmd-shift-p` on macOS, `ctrl-shift-p` on Linux)
3. Run `zed: extensions`
4. Search for `Nil UI`
5. Click `Install`

### As a dev extension

1. Clone this repository:
   ```sh
   git clone https://github.com/Nitrino/nil-ui-zed.git
   ```
2. Open Zed and run `zed: install dev extension` from the command palette
3. Select the cloned directory

## Activation

Once installed, open the command palette and run `theme selector: toggle`, then pick **Nil UI**.
